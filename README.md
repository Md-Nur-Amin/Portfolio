## Getting Started

### Prerequisites

- Node.js installed
- npm or yarn package manager

### Installation

```bash
git clone https://github.com/Md-Nur-Amin/Portfolio.git
cd your-repo
npm install
# or
yarn install
```


Running Locally:

npm run dev
# or
yarn dev
```
